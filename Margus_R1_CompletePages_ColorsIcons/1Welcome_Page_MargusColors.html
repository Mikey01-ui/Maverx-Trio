Game Overview

Welcome to The Data Game — a digital escape room built to turn data theory into real-world skill. Across five timed rounds, you'll inspect messy datasets, spot meaningful trends, make ethical calls, and connect insights to business processes. Your score rewards accuracy and speed, and every mistake triggers instant, constructive feedback so you learn while you play.

Five Rooms. One Skill Ladder.

Progress through five sequential challenges—each one unlocks only when you prove mastery. Build skills from data governance and analysis to ethics, process thinking, and decision storytelling.

Realistic Data Decisions

Work with imperfect information—conflicting datasets, ambiguous ownership, outliers, missing context. Learn how to judge reliability, select the right KPI, and separate signal from noise.

Score, Feedback, and Momentum

Your score reflects accuracy + time. Every action gets immediate feedback, so mistakes become learning moments. Progress saves automatically so you can return and finish later.
